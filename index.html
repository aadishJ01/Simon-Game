The Simon Game

Objective of the game:

Correctly repeat a longer sequence of signals.

Rules of the game:

1. The game consist of four colored buttons, each producing a particular tone when it is pressed.

2. In each round, a new button is lightened up in random order, after which the player must reproduce that order from the start by pressing the buttons.

3. If the player succeed in doing so, he will be promoted to the next level.

4. But, if the player failed to repeat the sequence exactly, the game will be ended.
